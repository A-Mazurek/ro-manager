Annotations (2)

  • comment "Generalized rule consisting of a query to extract information from an RDF graph, and a test that is applied to the results from applying that query. The query is assumed to return a set of results, each of which contains one ormore variable bindings. This currently available implementation uses a SPARQL query, but in principle other forms of query are possible, such as an OWL class expression. If a minim:graph value is supplied, it supplies a URI template used to identify the graph to be queried, otherwise the supplied RO annotation graph is queried." ()
