The results returned are a set of results, where each result is a set of query variable bindings corresponding to a single match of the query, and the entire result set contains all distinct matches achieved. The minim:orderBy value, if present, is added as a query result modifier, and may be used to obtain results returned in a consistent oder for better diagnostics." ()
